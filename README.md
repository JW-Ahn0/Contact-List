## input 요소 유효성 검사 - 한글만 입력, 핸드폰 형식에 맞춰서 입력, 빈 내용 체크

구현완료

한글의 경우 이름 중복이 있을 수 있으니 2자 이상 + 숫자도 허용하는 식으로 구현
ex) 안준우1, 안준우2 가능

핸드폰 형식의 경우 010-0000-0000 형식이어야 하며
하이푼(-) 의 경우 그냥 숫자를 입력하면 자동으로 달리도록 하였음.

빈 내용인 경우 '저장' 버튼 누를시에 alert 창 뜨도록 구현
